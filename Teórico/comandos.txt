
* var nomevariavel
- Para deixar explicito para melhor leitura de código que isso é uma declaração de variavel.


* alert(variavel)
- Abrir um dialog, dispara uma caixa de mensagem ;
- Utilizado mais em debug, não muito em produção.


* document.write(variavel)
- Representa o DOM ;
- Mostra a execução diretamente no browser.


* console.log()
- Muito usado em debugs ; 
- Para acompanhar o processo de uma execução de script ;
- Expõe os dados das variaveis no console do browser


* prompt()
- Abre um dialog mas com entrada para texto ;
- ex:
var nome = prompt('Digite seu nome:')


* IF / ELSE
if (condição) {
    - código que será executado

} else {
    - código que será executado
}
